Частные производные

Определение: Частная производная функции двух переменных по одному из её аргументов равна пределу отношения частного приращения функции к вызвавшему его приращению аргумента, когда приращение аргумента стремится к нулю

выберите точку, для которой хотите найти производную:     ($x_0$: $y_0$: )

Примеры

$$ f(x,y) = x+y-(x^2+y^2)^{1/2}$$
$$ f(x,y) = sin(2x-3y)$$
$$ f(x,y) = x sin(y) + y cos(x)$$
$$ f(x,y) = xye^{xy}$$
Ответ:
значение производной по x:
значение производной по y:


Решение

$f^{'}_x=\lim\limits_{\Delta x\to 0} \frac{f(x_0+{\Delta x},y_0)-f(x_0,y_0)}{\Delta x} = 1-\frac{x}{\sqrt{x^2+y^2}}$
$f^{'}_y=\lim\limits_{\Delta y\to 0} \frac{f(x_0,y_0+\Delta y)-f(x_0,y_0)}{\Delta y}= 1-\frac{y}{\sqrt{x^2+y^2}}$

Решение

$f^{'}_x=\lim\limits_{\Delta x\to 0} \frac{f(x_0+{\Delta x},y_0)-f(x_0,y_0)}{\Delta x} = 2cos(2x-3y)$
$f^{'}_y=\lim\limits_{\Delta y\to 0} \frac{f(x_0,y_0+\Delta y)-f(x_0,y_0)}{\Delta y}= -3cos(2x-3y)$

Решение

$f^{'}_x=\lim\limits_{\Delta x\to 0} \frac{f(x_0+{\Delta x},y_0)-f(x_0,y_0)}{\Delta x}=sin(y)-ysin(x)$
$f^{'}_y=\lim\limits_{\Delta y\to 0} \frac{f(x_0,y_0+\Delta y)-f(x_0,y_0)}{\Delta y}\frac{\partial f}{\partial y}=xcos(y)+cos(x)$

Решение

$f^{'}_x=\lim\limits_{\Delta x\to 0} \frac{f(x_0+{\Delta x},y_0)-f(x_0,y_0)}{\Delta x}=xe^{xy}(xy+1)$
$f^{'}_y=\lim\limits_{\Delta y\to 0} \frac{f(x_0,y_0+\Delta y)-f(x_0,y_0)}{\Delta y}\frac{\partial f}{\partial y}=ye^{xy}(xy+1)$
